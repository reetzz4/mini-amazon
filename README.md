# Mini Amazon (Backend)

This is a simple e-commerce backend project built using Node.js, Express, and MongoDB.

## 🚀 Features
- Product management (Add, View, Update, Delete)
- User management
- REST API structure
- MongoDB database using Mongoose

## 🛠️ Technologies Used
- Node.js
- Express.js
- MongoDB
- Mongoose

## 📁 Project Structure
- product/ → product model, controller, validation
- user/ → user model and controller
- middleware/ → custom middleware
- db.connection.js → database connection
- index.js → main entry file

## ▶️ How to Run
```bash
npm install
node index.js
